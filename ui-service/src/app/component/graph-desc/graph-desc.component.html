Graph Information
  • The paths shown in the graph have been found connecting the input subject and object
  • Click on a path to highlight it's explanation displayed in the right side-panel
  • Path score of each path is displayed as title of it's detail card in the right side-panel
  • Expand the path detail cards to view more details
  • Click on a path detail card to highlight its associated path in graph
  • The path score is obtained by calculating association measure (NPMI) between the path and the input property in a given KG
